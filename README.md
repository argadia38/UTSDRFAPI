## Cara menjalankan
- buka terminal  : comandprompt 
- aktifkan environment  :
  ```
  env\Script\activate
  ```
- masuk folder project  :
  ```
  cd blog
  ```
- jalankan server :
  ```
  python manage.py runserver
  ```
